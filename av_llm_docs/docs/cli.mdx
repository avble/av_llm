## usage

```shell
$ av_llm [command]
$ av_llm [usage]
```

## Commands

### chat

start an interactive chat

```
$ av_llm chat <model_path>
```

### server

start a server to serve the llm inference

```
$ av_llm serve <model_path>
```

### model

#### list

```shell
$ av_llm model ls

```

#### pull

```shell
$ av_llm model pull <model_path>
```

#### del

```shell
$ av_llm model del <model_path>
```

## flags

- -h: show help
- -v: show version
